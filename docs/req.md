# Функциональные требования
* В приложении должны агрегироваться и отображаться данные о философах из схемы.
* Отображение данных на временной линии
  * Отображение:
    * Философы (годы жизни)
    * Школы
    * Эпохи
  * Таймлайн
    * По оси x - шкала времени
    * Прокрутка вверх/вниз
    * Прокрутка по временной шкале
    * Ограниченное изменение масштаба по временной шкале
      * Размер шрифта не должен изменяться
  * Фильтры
    * Фильтр по имени
    * Границы таймлайна
    * Отобразить топ-N из Google Trends
* Отображение данных на графе
  * Отображение
    * Взаимное влияние философов
    * Отображение связей между школами, понятиями и философами
      * Несколько уровней:
        * Верхний уровень - школы, философы
        * Граф для школы
        * Граф для философа
          * Центрирован на одном философе
          * Отображены:
            * Понятия центрального философа
            * Прочие философы, работающие над этими понятиями
            * Школы философа
            * Прочие философы школ
          * Чекбоксы для отображения других философов
        * Переход к карточке понятия по нажатию на понятие
   * Навбар с кнопками:
     * Вперед/назад
     * Фильтры
* Карточки:
  * Субъекты:
    * Философ
    * Школа
    * Понятие
    * Эпоха
  * Навигация по карточкам
* Фильтры таймлайна, графа:
  * Поиск по строке по всем именам вершин
    * Все найденные вершины подсвечиваются
    * Навбар заменяется на навбар поиска с кнопками:
      * Вперед/назад по результатам
      * Завершить поиск
  * Отображение части:
    * По временному промежутку
* Глобальный поиск
  * Результаты - список карточек
